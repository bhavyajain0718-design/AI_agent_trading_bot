# AI\_agent\_trading\_bot

[Permalink: AI_agent_trading_bot](https://github.com/bhavyajain0718-design/AI_agent_trading_bot#ai_agent_trading_bot)

Autonomous AI trading agent that analyzes real-time market data, generates signals using RSI/EMA/volatility, and executes trades with dynamic risk management. Built with Python, Pandas, NumPy, and Kraken API/CLI, with optional ERC-8004 support for trustless on-chain execution.

## About

Autonomous AI trading agent that analyzes real-time market data, generates signals using RSI/EMA/volatility, and executes trades with dynamic risk management. Built with Python, Pandas, NumPy, and Kraken API/CLI, with optional ERC-8004 support for trustless on-chain execution.
